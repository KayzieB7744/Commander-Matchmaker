# Commander Matchmaker Starter App

This is a simple phone-friendly web app / PWA built from your Commander precon data.

## Files
- `index.html`
- `style.css`
- `app.js`
- `decks.json`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

## Run locally
Open the folder with a simple local web server.

Python:
```bash
python3 -m http.server 8000
```

Then open:
```text
http://localhost:8000
```

## Deploy free
GitHub Pages, Netlify, or Vercel all work well.

## iPhone home screen
Open the deployed site in Safari, then:
Share → Add to Home Screen
